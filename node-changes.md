# 节点规则改进
## 机器节点
* 记账节点
  - 负责出块，存储交易数据；
  - 至少锁定5000万NEW；
  - 数量19个正选节点，备选节点无限制；
  - 权益
     - 节点锁定牛力加速系数：4
     - 投票锁定牛力加速系数：1
     - 开发DApp
